# Project 1 - Password Strength Checker

A simple password strength checker built as part of the DecodeLabs Cybersecurity Internship (Batch 2026).

---

## About the Project

This project checks how strong a password is based on a set of rules. It gives instant feedback as the user types, showing which conditions are met and the overall strength level.

---

## Features

- Live password strength checking as you type
- Show / Hide password toggle
- Strength label — Weak / Fair / Good / Strong / Very Strong
- Checklist of 5 rules with ✗ / ✓ indicators

---

## Password Rules

| Rule | Description |
|------|-------------|
| Length | At least 8 characters |
| Uppercase | At least one uppercase letter (A-Z) |
| Lowercase | At least one lowercase letter (a-z) |
| Number | At least one number (0-9) |
| Special Character | At least one special character (!@#$...) |
